Chat App

My Firebase Chat app is purely developed using the Android with Java language. Its a real-time chatting app between two people using the Firebase Database. The app has premium features that enhance the user experience. Users can send messages, files, and voice; be able to see online or offline status; type indicators; support emojis; and display date/time-wise messages. This app uses Firebase as a backend part.
