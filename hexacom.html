Hexacom

Single vendor eCommerce Flutter App for Android & iOS with Website, Laravel Admin Panel and Delivery boy app

App Features
  • User App Features
    • Multi-branch supported
    • Coverage area
    • Self Pickup & Home delivery
    • Splash & OnBoarding screen
    • Login and Registration with email & password
    • Password reset option using confirmation OTP
    • Multiple languages
    • LTR & RTL supported
    • Smart homepage with category, Offer Products, banner & Popular items
    • Promotional banner option with category or item redirection
    • Nice category & subcategory page
    • Search product item with search suggestions
    • Filter option in the search
    • Product wishlist
    • Cart screen with the dynamic configurable payment method
    • Promo code system
    • Order list and details
    • Order tracking with live delivery man location
    • Save addresses for quick order
    • In-app chatting with the owner
    • Custom notification
    • User profile
    • Dark & Light theme for better user experience
  • Delivery App Features:
    • Splash & Login with email and password
    • Active order list
    • Order details with Start delivery and Confirm delivery button
    • Delivery success screen with the receivable amount from the customer
    • Google map direction for customer delivery location
    • Delivered order history
    • Delivery man profile
    • LTR & RTL supported
    • Dark & Light theme
  • Admin Dashboard Features:
    • Branch management
    • Order management
    • Product, category & subcategory management
    • Chat with customer
    • Send custom notification
    • Coupon management
    • Customer management
    • Delivery man management
    • Rich Analytics & Reports
    • Business settings
