App Features
  • Customer App Features:
    • Easy to Sign up & Sign in with Mobile Number Verification
    • Sign in with Mobile number/Email
    • App Home screen product and category managed by admin
    • Product Variations with All the type.
    • Product Details with Multiple Images and Product Description
    • Easy to Add to cart and Orders
    • User friendly Date & time and Payment Gateway(Pickup Myself, Cash on Delivery (COD), PayPal, Razorpay)
    • Instant Notification for Product Add or Any Seasonal Offer with Image
    • Add / update / delete Multiple Address
    • Update profile anytime
    • Check order tracking status with Pending, Ready to Ship or Delivered
    • Check all order details info
    • Rate the Product with Very like it, Good or not good
    • Contact delivery boy once the order proceed
    • Cancel the order
    • Provide app or product feedback
    • Share app with friends and family
    • Check Privacy Policy, About us, Contact us, Terms & Condition page
    • Check Notification & Notification Details
    • Change the Forget Password with Mobile OTP verification
    • Check the order details status Complete, Processing, Pending and history of orders
    • Receive notification once the product Accepted, Delivered
  • Delivery Boy App Features:
    • Login with mobile number
    • Check all the pending order / delivered order list and details
    • Accept or reject the order
    • Track the order
    • Cancel the order for reason like Issue with on-going order, Payment issue with my order, Address wrong, other
    • Contact the order person with mobile number dial-up
    • Open instant map
    • Deliver product with proof of customer/user signature (Signed proof of delivery)
    • Check all the notifications
    • Check profile and update the details anytime
    • Change status with active or inactive
    • Check the Order completed, Total Sales and Cancel order statistics (insight)
    • Receive notification once the admin assigns the new orders
    • Accept/deliver the order makes the user get a notification
